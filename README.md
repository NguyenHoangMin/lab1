# Lab1 - ITA105

Dự án xử lý dữ liệu với Python sử dụng pandas và matplotlib.

## Mô tả
Dự án Lab1 bao gồm:
- **Bài 1**: Khám phá dữ liệu (shape, null values)
- **Bài 2**: Xử lý dữ liệu thiếu (fillna)
- **Bài 3**: Xử lý dữ liệu lỗi (filtering)
- **Bài 4**: Làm mượt dữ liệu và vẽ biểu đồ (moving average)
- **Bài 5**: Chuẩn hóa dữ liệu (normalization)
